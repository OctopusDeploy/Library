Chocolatey - Install Package

Octopus.Script exported 05/06/2014 by nblumhardt

Installs a package using the Chocolatey package manager.

Parameters

When steps based on the template are included in a project's deployment process, the parameters below can be set.

Chocolatey package

#{ChocolateyPackageId} =

The ID of an application package in the https://chocolatey.org repository. Required. Example: Git.

Version

#{ChocolateyPackageVersion} =

If a specific version of the Chocolatey package is required enter it here. Otherwise, leave this field blank to use the latest version. Example: 2.3.4.

Script body

Steps based on the template will execute the following PowerShell script.

$chocolateyBin = "$env:SystemPath\\chocolatey\\bin\\cinst.bat"if (-not $ChocolateyPackageId) {    throw "Please specify the ID of an application package to install."}if (-not (Test-Path $chocolateyBin)) {    throw "Chocolatey was not found at $chocolateyBin."}if (-not $ChocolateyPackageVersion) {    Write-Output "Installing package $ChocolateyPackageId from the Chocolatey package repository..."    & $chocolateyBin $ChocolateyPackageId} else {    Write-Output "Installing package $ChocolateyPackageId version $ChocolateyPackageVersion from the Chocolatey package repository..."    & $chocolateyBin $ChocolateyPackageId -Version $ChocolateyPackageVersion}
